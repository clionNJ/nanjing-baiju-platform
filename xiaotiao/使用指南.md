# 南京白局非遗文化数字平台 - 使用指南

## 📋 项目简介

这是一个用于展示和传承南京白局（非物质文化遗产）的数字化平台。平台通过虚拟技术、AI技术等手段，让用户能够：
- 观看虚拟传承人的表演
- 进行AI语音跟唱训练
- 在虚拟课堂中学习白局知识
- 浏览白局文化知识库
- 使用AI创作白局作品
- 提交反馈和建议

## 🚀 快速开始

### 1. 环境要求
- 现代浏览器（Chrome、Firefox、Edge、Safari等）
- 无需安装任何服务器或数据库
- 支持本地文件直接打开

### 2. 运行方式

#### 方式一：直接打开（最简单）
1. 找到项目文件夹中的 `index.html` 文件
2. 双击打开，浏览器会自动加载页面

#### 方式二：使用本地服务器（推荐）
```bash
# 如果安装了Python
python -m http.server 8000

# 如果安装了Node.js
npx http-server

# 然后在浏览器访问 http://localhost:8000
```

### 3. 文件结构
```
xiaotiao/
├── index.html          # 主页面文件（包含所有代码）
├── images/             # 图片资源文件夹
│   ├── 4ED9557E923B0B49B6F4A467C16AB514.jpg  # 背景图片
│   └── 3DB4A5D6F35C44B34ECBC88BD7D20D37.jpg  # 虚拟人物图片
├── 白局小明视频.mp4    # 虚拟传承人视频
└── 使用指南.md         # 本文件
```

## 📖 功能使用说明

### 第一步：选择语言
1. 打开页面后，首先看到语言选择界面
2. 点击您偏好的语言卡片（中文、English、Español、Français、日本語、العربية）
3. 页面会自动切换到功能主界面

### 第二步：探索功能模块

#### 🎭 功能一：虚拟传承人演唱展示
1. 点击"虚拟传承人演唱展示"卡片
2. 在弹出的详情窗口中：
   - 点击"播放《秦淮灯会》"或"播放《金陵四季》"按钮开始播放视频
   - 点击"停止播放"按钮停止视频
   - 也可以先选择唱段卡片，再点击播放按钮
3. 视频会自动循环播放

#### 🎤 功能二：AI语音跟唱训练
1. 点击"AI语音跟唱训练"卡片
2. 选择要练习的唱段（秦淮灯会、金陵四季、南京美食）
3. 查看显示的歌词
4. 点击"开始跟唱"按钮
5. 系统会模拟评分过程，显示：
   - 音准评分
   - 节奏评分
   - 发音评分
6. 歌词行会根据跟唱情况显示绿色（正确）或红色（错误）

#### 🏫 功能三：虚拟课堂沉浸式教学
1. 点击"虚拟课堂沉浸式教学"卡片
2. 选择课程类型：
   - 白局历史
   - 唱腔技巧
   - 乐器介绍
3. 在输入框中输入问题，点击"提问"按钮（功能可扩展）

#### 📚 功能四：白局知识库与文化展厅
1. 点击"白局知识库与文化展厅"卡片
2. 浏览知识卡片：
   - 历史渊源
   - 代表人物
   - 乐器介绍
   - 服饰文化
3. 点击"了解更多"按钮查看详细信息
4. 可以切换语言查看不同语言版本的内容

#### ✨ 功能五：AI创作与分享中心
1. 点击"AI创作与分享中心"卡片
2. 在输入框中输入创作主题（如：南京美食、龙舟节、金陵秋色）
3. 选择输出语言
4. 点击"生成白局作品"按钮
5. 查看AI生成的唱词
6. 可以点击"试听作品"或"分享到社交平台"（功能可扩展）

#### 💬 功能六：意见与反馈
1. 点击"意见与反馈"卡片
2. 填写反馈表单：
   - 选择反馈类型（功能建议、问题反馈、内容建议、其他）
   - 输入标题
   - 输入详细内容
   - 填写联系方式（选填）
   - 选择满意度评分（1-5星）
3. 点击"提交反馈"按钮
4. 系统会显示提交成功提示

### 第三步：返回和切换
- 点击页面顶部的"切换语言"可以返回语言选择页面
- 点击"返回语言选择"按钮也可以返回
- 点击"获取帮助"查看帮助信息

## 🔧 常见问题

### Q1: 背景图片不显示？
**A**: 检查 `images` 文件夹中是否存在对应的图片文件，路径是否正确。

### Q2: 视频无法播放？
**A**: 
- 检查视频文件 `白局小明视频.mp4` 是否在项目根目录
- 检查浏览器是否支持MP4格式
- 尝试使用不同的浏览器

### Q3: 页面样式错乱？
**A**: 
- 检查网络连接，确保Font Awesome图标库能正常加载
- 清除浏览器缓存后重新加载
- 检查浏览器控制台是否有错误信息

### Q4: 功能按钮无响应？
**A**: 
- 打开浏览器开发者工具（F12）查看控制台错误
- 确保JavaScript已启用
- 检查是否有浏览器扩展阻止了脚本运行

## 📝 开发说明

### 技术栈
- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript**：交互逻辑
- **Font Awesome**：图标库（CDN）

### 代码结构
- 所有代码都在 `index.html` 文件中
- CSS样式在 `<style>` 标签中
- JavaScript代码在 `<script>` 标签中
- 使用单页面应用（SPA）架构

### 扩展建议
