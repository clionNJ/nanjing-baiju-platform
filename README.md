# 南京白局非遗文化数字平台

> 一个用于展示和传承南京白局（非物质文化遗产）的数字化平台

## 🌟 项目简介

本项目通过数字化手段，结合虚拟技术、AI技术等现代科技，为南京白局这一非物质文化遗产提供：
- 虚拟传承人演唱展示
- AI语音跟唱训练
- 虚拟课堂沉浸式教学
- 白局知识库与文化展厅
- AI创作与分享中心
- 意见与反馈系统

## ✨ 主要功能

- 🎭 **虚拟传承人演唱展示** - 观看数字化还原的白局传承人表演
- 🎤 **AI语音跟唱训练** - 实时分析音高、节奏、发音，提供专业指导
- 🏫 **虚拟课堂沉浸式教学** - 在虚拟场景中学习白局历史与唱腔技巧
- 📚 **白局知识库与文化展厅** - 了解白局起源、发展、代表人物等丰富文化知识
- ✨ **AI创作与分享中心** - AI自动生成白局唱词与旋律
- 💬 **意见与反馈** - 收集用户建议，持续改进平台

## 🌍 多语言支持

支持6种语言界面：
- 🇨🇳 中文
- 🇺🇸 English
- 🇪🇸 Español
- 🇫🇷 Français
- 🇯🇵 日本語
- 🇸🇦 العربية

## 🚀 快速开始

### 本地运行

1. **直接打开**
   - 双击 `index.html` 文件即可在浏览器中打开

2. **使用本地服务器（推荐）**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   
   # 然后访问 http://localhost:8000
   ```

### 部署到服务器

详细部署指南请查看 [部署指南.md](./部署指南.md)

**快速部署（GitHub Pages）：**
1. 将项目上传到GitHub仓库
2. 在仓库 Settings → Pages 中启用
3. 选择 main 分支作为源
4. 等待几分钟，访问 `https://YOUR_USERNAME.github.io/REPO_NAME`
